# Country Search

This program uses PHP 7.3 and React 16

### To run:

(After cloning repo)
```
$ ./server
```

Then go to:
```
http://localhost:8765
```

### To run in development mode:

```
$ cd ./react
$ npm install
$ npm start
```

And visit:
```
http://localhost:3000
```

### To build for production:

```
$ cd ./react
$ npm run build
```
And then follow instructions in "To run" section above.
